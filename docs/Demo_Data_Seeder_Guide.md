## 🚀 كيفية الاستخدام

### طريقة 1: من خلال الواجهة (UI)
1. افتح النظام كـ **SuperAdmin**
2. من القائمة الجانبية → **أدوات التطوير** → **مولد البيانات التجريبية**
3. اضغط على **"إنشاء البيانات التجريبية"**
4. شاهد التقرير المباشر في الواجهة

### طريقة 2: من خلال Console
```javascript
// إنشاء البيانات
DemoSeeder.seed();

// حذف البيانات
DemoSeeder.cleanup();

// عرض الإحصائيات
DemoSeeder.getStats();
```

## 🔐 الأمان

### ✅ البيانات آمنة تماماً
- جميع البيانات التجريبية معلّمة بـ `is_demo: true`
- لن تتداخل مع البيانات الحقيقية
- قابلة للحذف بضغطة زر واحدة

### 🧹 حذف البيانات
```javascript
DemoSeeder.cleanup();
```
سيحذف **فقط** البيانات المعلّمة بـ `is_demo: true`

## 🧪 حالات الاختبار المقترحة

### 1. **اختبار لوحة التحكم**
- التحقق من ظهور الإحصائيات الصحيحة
- فلترة البيانات حسب الفرع
- عرض الرسوم البيانية

### 2. **اختبار إدارة الطلاب**
- عرض قائمة الطلاب
- عرض تفاصيل الطالب
- البحث عن طالب

### 3. **اختبار المخزون**
- عرض الكتب المتاحة
- التحقق من الكميات
- تنبيهات المخزون المنخفض

### 4. **اختبار المحاسبة**
- عرض القيود المحاسبية
- تقارير الأرباح والخسائر
- رصيد الطلاب

### 5. **اختبار الفلترة حسب الفرع**
- اختر "فرع 1" من القائمة العلوية
- تحقق من ظهور البيانات التجريبية
- جرب "جميع الفروع" وتحقق من التجميع

## ⚠️ ملاحظات مهمة

### 1. **الفرع الافتراضي**: جميع البيانات تُنشأ في **فرع 4**
   - إذا لم يكن موجوداً، سيتم إنشاؤه تلقائياً

2. **القيود المحاسبية**: يتم إنشاءها تلقائياً لـ:
   - المدفوعات (Debit: Cash / Credit: Revenue)
   - المشتريات (Debit: Inventory / Credit: Cash)

3. **عدم التكرار**: يمكنك تشغيل الأداة عدة مرات
   - ستُنشئ بيانات جديدة بـ IDs مختلفة
   - لن تُحذف البيانات القديمة تلقائياً

4. **الحذف الآمن**: `cleanup()` يحذف فقط البيانات المعلّمة بـ `is_demo: true`

## 🎓 أمثلة استخدام

### مثال 1: اختبار التقارير المالية
```javascript
// أنشئ البيانات
DemoSeeder.seed();

// افتح صفحة "الذكاء المالي"
// شاهد:
// - إجمالي الإيرادات: 5,500 ج.م
// - إجمالي المصروفات: (تكلفة المخزون)
// - صافي الربح
```

### مثال 2: اختبار فلترة الفروع
```javascript
// أنشئ البيانات في فرع 4
DemoSeeder.seed();

// من القائمة العلوية:
// - اختر "فرع 4" → شاهد 5 طلاب
// - اختر "جميع الفروع" → شاهد جميع الطلاب
```

### مثال 3: اختبار وحذف
```javascript
// إنشاء
DemoSeeder.seed();

// الاختبار...
// (قم باختبار الميزات)

// الحذف
DemoSeeder.cleanup();
```
