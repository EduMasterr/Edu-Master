# Scope of Work (نطاق العمل) - EduMaster Pro
**المرحلة:** 1 (التخطيط والتحليل)
**التاريخ:** 2026-02-11

---

## 1. المقدمة (Introduction)
المشروع عبارة عن **نظام إدارة معاهد تعليمية شامل (EduMaster Pro)**، يخدم شبكة من المعاهد ذات النشاطات المتعددة (لغات، برمجة، أطفال، تدريب مهني). النظام مصمم ليدعم (8 فروع) مبدئياً، مع بنية قابلة للتوسع.

---

## 2. النطاق الداخلي (In-Scope)
- **إدارة المستخدمين:** جميع الأدوار (Super Admin, Branch Admin, Teacher, Employee, Student, Parent).
- **إدارة الفروع (Multi-branch):** لوحة تحكم لكل فرع ولوحة تحكم مركزية.
- **إدارة الطلاب:** ملف إلكتروني شامل لكل طالب (صور، مستندات، بيانات).
- **إدارة الكورسات والمجموعات:** هيكل كورسات هرمي، جداول، وحصص.
- **إدارة الحضور والغياب:** تسجيل يدوي وتلقائي للحضور لكل مجموعة.
- **إدارة المدفوعات والاشتراكات:** نظام محاسبي بسيط ومتقدم، فواتير، وتقارير.
- **نظام تقارير (Report Generator):** تقارير شهرية وسنوية لجميع الجوانب.
- **واجهة المستخدم (UI/UX):** تصميم عصري (Dark Mode) وResponsive (موبايل/تابلت).

---

## 3. النطاق الخارجي (Out-of-Scope)
- **تطبيق موبايل (Native Mobile App):** التطبيق سيكون Web App (PWA) وليس Native.
- **التكامل مع بوابات دفع إلكتروني (Payment Gatways):** مبدئياً خارج النطاق (Placeholders).
- **الذكاء الاصطناعي (AI Analysis):** غير مشمول في المرحلة الأولى (يمكن إضافته لاحقاً).
- **نظام التعليم عن بعد المتكامل (Full LMS - Video Conferencing):** سيتم استخدام روابط خارجية (Zoom/Google Meet) بدلاً من استضافة الفيديو.
